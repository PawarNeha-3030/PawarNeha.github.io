🌾 FarmAI Assistant - Smart Farming Solution

## About This Project
This is an AI-powered farming assistant that helps farmers get crop recommendations, water planning, and disease alerts based on their location and soil type.

## Features
1. 📍 Location Detection
2. 🌱 Crop Recommendations  
3. 💧 Water Calculator
4. ⚠️ Disease Alerts
5. 📅 Farming Calendar
